\# SQL Practice Queries (Employee Database)



\## Introduction

This repository contains a collection of 55 SQL practice queries created using an Employee database schema.  

The queries are designed to improve understanding of SQL concepts from basic filtering to advanced conditional operations.



It is useful for:



\- SQL practice

\- Interview preparation

\- Academic learning

\- Strengthening database query skills





\# Database Structure



\## Employees Table



\- emp\_id

\- emp\_name

\- department

\- salary

\- city

\- experience





\# Topics Covered



\## Basic SQL Queries

\- Selecting all columns using `SELECT \*`

\- Selecting specific columns

\- Filtering records using `WHERE`



\## Aggregate Functions

\- `SUM()` for total salary

\- `AVG()` for average salary

\- `COUNT()` for counting employees

\- `MAX()` for highest values

\- `MIN()` for lowest values



\## Grouping Data

\- Using `GROUP BY`

\- Filtering grouped data using `HAVING`



\## Sorting and Limiting Data

\- Sorting records using `ORDER BY`

\- Displaying top records using `LIMIT`



\## DISTINCT Keyword

\- Retrieving unique department names

\- Retrieving unique city names

\- Retrieving distinct combinations



\## Comparison Operators

\- `>`

\- `<`

\- `>=`

\- `<=`

\- `=`

\- `<>`



\## Logical Operators

\- `AND`

\- `OR`

\- `NOT`



\## IN and NOT IN

\- Matching values from a list

\- Excluding values from a list



\## BETWEEN Operator

\- Selecting values within a range

\- Using `NOT BETWEEN`



\## LIKE Operator

\- Pattern matching using `%`

\- Finding names starting or ending with characters
